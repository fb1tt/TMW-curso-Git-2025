git init . -> cria repo git na pasta atual 
status -> mostra status do trabalho (de cada arquivo 
git add nome_arquivo -> adiciona arquivo para se commitado
git commit -m "msg" -> faz commit do que estava paa ser commitado
git log -> mostra todo o histórico de commit

## Fluxo de trabalho

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
